# ☕ Afficionado Coffee Roasters Dashboard

## Project Overview

This project analyzes coffee shop sales data and provides interactive business insights using Streamlit. It also demonstrates demand forecasting using the Prophet time-series forecasting model.

## Features

- Interactive Dashboard
- Revenue Analysis
- Store-wise Analysis
- Product Category Analysis
- Peak Sales Hours
- Demand Forecasting
- Business Insights

## Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
- Prophet

## Dataset

Afficionado Coffee Roasters Sales Dataset

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
